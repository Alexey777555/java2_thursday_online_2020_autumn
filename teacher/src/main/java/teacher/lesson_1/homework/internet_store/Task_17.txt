Перепешите методы удаления и изменения данных на работу с id.

Пользователь указывает id продукта, который хочет удалить,
программа находит и удаляет продукт с указанным id.

Пользователь указывает id продукта, у которой хочет изменить
название или описание, программа находит продукт с указанным id
и меняет у него свойства.
