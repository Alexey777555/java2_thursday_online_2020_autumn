Создайте класс BookLibraryApplication с main() методом.

Напишите консольное приложение со следующей функциональностью:
1. приложение выводит на консоль доступные пункты меню
   (например: 1 - добавить книгу, 2 - удалить книгу, 3 - список книг, 4 - выход из программы);
2. пользователь выбирает пункт меню, который он хочет выполнить;
3. программа выполняет указанный пункт меню;
4. программма снова показывает меню;
5. если пользователь выбирает пункт меню выйти из программы
   то программа завершает свою работу.

На данном этапе создаваемые объекты храните в памяти программы,
используйте для этого например структуру данных список.

Задачу можно разделить между членами команды,
каждый реализует свой пункт меню. Если необходимо
вы можете расширить функциональность вашего приложения
добавив например поиск по автору, названию, изменение
автора или названия уже добавленной книги.
