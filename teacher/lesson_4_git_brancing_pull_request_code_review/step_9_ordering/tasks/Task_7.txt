Функциональность сортировки является независимой от поиска книг
и может быть применена так же в другом месте. Например при
сортировке скиска читателей. Следуя принципу SRP нужно
реализовать сортировку по возможности отдельно.

В запросе SearchBooksRequest сортировка указывается
при помощи двух свойств:

  	private String orderBy;
  	private String orderDirection;

Создайте класс Ordering и перенесите в него свойства
касающиеся сортировки. Класс Ordering создайте в
пакете /core/requests.
