Так как список найденных в библиотеке книг может быть
довольно большим и его довольно трудно просматривать
необходимо в функционал поиска ввести возможность
указывать порядок сортировки списка найденных книг.
Для этого в класс SearchBooksRequest добавьте следуюшие поля:
    private String orderBy;
    private String orderDirection;
